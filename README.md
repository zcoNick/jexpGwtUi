# JavExpress

This is an overview page, please visit [JavExpress.org](http://www.javexpress.com/jexpgwtui.html) for more information.

![JavExpress icon](http://blog.javexpress.com/jexpgwtui/wp-content/uploads/2011/08/bloglogo.png)

### Overview
***

[JavExpress](http://www.javexpress.com/jexpgwtui/index) is one of the most popular GWT UI libraries in Java Ecosystem and widely used by software companies, world renowned brands, banks, financial institutions, insurance companies, universities and more. Here are [some of the users](http://www.javexpress.com/jexpgwtui/whouses) who notified us or subscribed to a [JavExpress Support Service](http://www.javexpress.com/jexpgwtui/support).

### Getting Started
***

**JavExpress** can be downloaded manually or via maven.

##### Maven

```xml
<dependency>  
    <groupId>com.javexpress</groupId>  
    <artifactId>jexp-common</artifactId>  
    <version>0.0.1</version>  
</dependency>  
```
